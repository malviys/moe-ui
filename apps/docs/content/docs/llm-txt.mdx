## What is llm.txt?

llm.txt is a standard format for providing documentation that's optimized for Large Language Models. It helps AI coding assistants understand your component library better.

## Planned Features

- **Structured documentation** - Machine-readable component docs
- **Usage examples** - Ready-to-use code snippets for LLMs
- **API reference** - Complete props and methods documentation
