## Create project

Run `init` to start a new project, or follow the manual steps to set up an existing one.

<Tabs items={["npm", "bun", "pnpm", "yarn"]}>
  <Tab value="npm">```bash npx @moe/cli@latest init ```</Tab>
  <Tab value="bun">```bash bunx --bun @moe/cli@latest init ```</Tab>
  <Tab value="pnpm">```bash pnpm dlx @moe/cli@latest init ```</Tab>
  <Tab value="yarn">```bash npx @moe/cli@latest init ```</Tab>
</Tabs>

This creates a new project in the current directory.

### Package Manager Options

You can specify which package manager to use for installing dependencies:

```bash
npx @moe/cli@latest init --pm bun
npx @moe/cli@latest init --pm npm
npx @moe/cli@latest init --pm pnpm
```

The CLI will use your specified package manager for all dependency installations.

## Add components

You can now start adding components to your app.

<Tabs items={["npm", "bun", "pnpm", "yarn"]}>
  <Tab value="npm">```bash npx @moe/cli@latest add button ```</Tab>
  <Tab value="bun">```bash bunx --bun @moe/cli@latest add button ```</Tab>
  <Tab value="pnpm">```bash pnpm dlx @moe/cli@latest add button ```</Tab>
  <Tab value="yarn">```bash npx @moe/cli@latest add button ```</Tab>
</Tabs>

You can also specify the package manager when adding components:

```bash
npx @moe/cli@latest add button --pm bun
```

The command above will add the `Button` component to your project. You can then import it like this:

```tsx title="index.tsx"
import { Button } from "@/components/ui/button";
import { Text } from "@/components/ui/text";

export default function Screen() {
  return (
    <Button>
      <Text>Click me</Text>
    </Button>
  );
}
```
